# ✨ Keerthi's Birthday Microsite — A Digital Celebration 🎂

A premium, immersive, and emotionally resonant birthday microsite designed specifically for Keerthi. This project transforms a standard greeting into a cinematic digital journey filled with interactive memories, animations, and heartfelt messages.

## 🌟 The Experience
The microsite follows a bespoke user journey designed to "WOW" from the first second:
1.  **Cinematic Welcome**: A heartbeat-led entry that transitions into a slow-paced keyword montage (Trust, Laughter, Loyalty, etc.).
2.  **Secret Portal**: A bespoke login gate requiring specific credentials to unlock the celebration.
3.  **Interactive Dashboard**: A high-end UI featuring:
    *   **Teddy Cursor**: A custom-animated 🧸 pointer with a magical sticker trail (🎂, 🎁, ✨).
    *   **Infinite Memories**: A 3D tilt-and-glow horizontal gallery of shared moments.
    *   **Magical Bestie Tree**: A procedural "growing tree" layout for the bucket list that sprouts as you scroll.
    *   **Interactive Letter**: A digital envelope that "types out" a heartfelt message in real-time.

## 🛠️ Tech Stack
*   **Structure**: Semantic HTML5
*   **Styling**: Vanilla CSS (Glassmorphism, 3D Transforms, Custom Animations)
*   **Interactions**: GSAP (GreenSock Animation Platform)
*   **Smooth Scrolling**: Lenis Scroll
*   **Visual Effects**: Canvas-Confetti & Procedural JS Decorations
